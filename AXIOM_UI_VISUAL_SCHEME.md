## 🖱️ Интерактивные элементы

### Главное меню
```
Элемент                 Действие
─────────────────────────────────────────────────
[Кнопка категории]  →  Фильтр по категории
[Колесо мыши]       →  Прокрутка списка
[▶ Выполнить]       →  Открыть форму ввода
[ℹ Подробнее]       →  Показать детали
[🔍 Поиск]          →  Поиск команд
[◀ Назад]           →  Закрыть меню
```

### Экран деталей
```
Элемент                 Действие
─────────────────────────────────────────────────
[◀ Назад]           →  Вернуться в меню
[▶ Выполнить]       →  Открыть форму ввода
```

### Форма ввода
```
Элемент                 Действие
─────────────────────────────────────────────────
[Поле ввода]        →  Редактировать команду
[Enter]             →  Выполнить команду
[✓ Выполнить]       →  Выполнить команду
[✗ Отмена]          →  Вернуться назад
```
