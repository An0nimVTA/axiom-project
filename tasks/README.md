# Tasks Hub

Назначение: хранить отдельные задачи с контекстом, ссылками на документацию и требованиями.

Структура:
- `tasks/INDEX.md` — список задач.
- `tasks/TEMPLATE.md` — шаблон новой задачи.
- `tasks/task-XXX-*.md` — сами задачи.

Правила:
- Одна задача = один файл.
- В задаче обязательно: цель, ссылки на документы, краткие знания/контекст, шаги, критерии готовности.
- Имена файлов: `task-001-<slug>.md`.
