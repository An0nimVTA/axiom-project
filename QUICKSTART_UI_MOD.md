# 🎮 AXIOM UI Mod - Быстрый старт

## ⚡ Установка за 3 шага

### 1️⃣ Сборка и установка
```bash
./build-ui-mod.sh
```

### 2️⃣ Запуск Minecraft
- Выбрать профиль **Forge 1.20.1**
- Подключиться к серверу с **AXIOM плагином**

### 3️⃣ Открыть меню
```
/axiomui
```

---

## 🎯 Что это?

**AXIOM UI Mod** - графический интерфейс для управления плагином AXIOM.

**Больше не нужно запоминать 100+ команд!**

---

## ✨ Возможности

✅ **40+ команд** в удобном UI  
✅ **9 цветовых категорий**  
✅ **Детальное описание** каждой команды  
✅ **Форма ввода** для параметров  
✅ **Прокрутка и фильтрация**  

---

## 📋 Категории

| Цвет | Категория | Примеры команд |
|------|-----------|----------------|
| 🟢 | **Нация** | `/nation create`, `/claim` |
| 🟡 | **Экономика** | `/wallet`, `/bank`, `/trade` |
| 🔵 | **Дипломатия** | `/treaty`, `/ally` |
| 🟣 | **Технологии** | `/tech`, `/tech research` |
| 🔴 | **Военное дело** | `/war declare`, `/capture` |
| 🔷 | **Города** | `/city create`, `/city build` |
| 🟠 | **Религия** | `/religion create` |
| ⚫ | **Профиль** | `/profile`, `/history` |
| ⚪ | **Админ** | `/axiom reload`, `/testbot` |

## 🐛 Проблемы?

### Мод не загружается
```bash
/forge mods  # Проверить список модов
```

### Проверить логи
```bash
tail -f ~/.minecraft/logs/latest.log | grep axiomui
```

### Пересобрать мод
```bash
./build-ui-mod.sh
```
