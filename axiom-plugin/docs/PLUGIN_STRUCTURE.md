# Структура проекта AXIOM плагина

## Директории

### src/main/java/com/axiom
- **command/** - команды плагина
- **expansion/** - расширения (PlaceholderAPI)
- **gui/** - графический интерфейс
- **listener/** - слушатели событий
- **model/** - модели данных
- **service/** - сервисы (165+ систем)
- **test/** - автотесты

### src/test/java
- **com/axiom/** - модульные тесты
- **integration/** - интеграционные тесты

### resources
- **config.yml** - основная конфигурация
- **lang/** - файлы локализации

## Тестирование

### Автобот
- `/testbot run` - запустить все тесты
- `/testbot run <тест>` - запустить конкретный тест
- `/testbot list` - список тестов

### Типы тестов
1. **Модульные** - тестирование отдельных классов
2. **Интеграционные** - тестирование взаимодействия систем
3. **Функциональные** - тестирование команд и функций
4. **Регрессионные** - проверка совместимости

## Системы
1. Экономика
2. Нации и территории
3. Дипломатия
4. Война и конфликты
