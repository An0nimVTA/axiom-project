# Команды AXIOM плагина

## Основные команды

### /axiom
**Описание:** Главная команда плагина
**Алиасы:** `/ax`, `/axiom`
**Разрешение:** `axiom.use` или `axiom.admin`
**Использование:** 
- `/axiom` - открыть главное меню
- `/axiom help` - показать список команд
- `/axiom version` - показать версию плагина
- `/axiom reload` - перезагрузить конфигурацию (требует `axiom.admin`)
- `/axiom test run all` - запустить все тесты (требует `axiom.admin`)
- `/axiom stats` - показать статистику сервера

## Национальные команды

### /nation
**Описание:** Управление нацией
**Алиасы:** `/n`, `/country`, `/nat`
**Разрешение:** `axiom.nation.use`
**Использование:**
- `/nation create <название>` - создать нацию
- `/nation info` - информация о вашей нации
- `/nation list` - список всех наций
- `/nation invite <игрок>` - пригласить игрока
- `/nation kick <игрок>` - исключить игрока
- `/nation setcapital` - установить столицу
- `/nation rename <новое_название>` - переименовать нацию (требует `axiom.nation.admin`)
- `/nation disband` - распустить нацию

### /claim
**Описание:** Захват территории для нации
**Алиасы:** `/n claim`, `/nation claim`
**Разрешение:** `axiom.nation.claim`
**Использование:**
- `/claim` - захватить текущий чанк для нации
- `/claim auto` - включить автоматический захват при перемещении
- `/claim cost` - показать стоимость захвата

### /unclaim
**Описание:** Освобождение территории нации
**Алиасы:** `/n unclaim`, `/nation unclaim`
**Разрешение:** `axiom.nation.unclaim`
**Использование:**
- `/unclaim` - освободить текущий чанк
- `/unclaim all` - освободить все территории (требует `axiom.nation.admin`)

## Экономические команды

### /wallet
**Описание:** Управление личным кошельком
**Алиасы:** `/money`, `/bal`, `/balance`
**Разрешение:** `axiom.economy.use`
**Использование:**
- `/wallet` - проверить баланс
- `/wallet <игрок>` - проверить баланс другого игрока (требует `axiom.economy.see`)
- `/wallet pay <игрок> <сумма>` - перевести деньги
- `/wallet top` - топ богачей

### /bank
**Описание:** Банковская система
**Алиасы:** `/banking`, `/deposit`, `/withdraw`
**Разрешение:** `axiom.bank.use`
**Использование:**
- `/bank` - открыть банковский интерфейс
- `/bank deposit <сумма>` - положить деньги в банк
- `/bank withdraw <сумма>` - снять деньги из банка
- `/bank balance` - проверить банковский баланс

### /trade
**Описание:** Система торговли
**Алиасы:** `/market`, `/exchange`
**Разрешение:** `axiom.trade.use`
**Использование:**
- `/trade` - открыть торговый интерфейс
- `/trade create <предмет> <цена>` - создать предложение
- `/trade buy <ID>` - купить предложение
- `/trade list` - список всех предложений

## Дипломатические команды

## Административные команды

**Описание:** Административные функции
**Разрешение:** `axiom.admin`
**Использование:**
- `/axiom admin give <игрок> <сумма>` - дать деньги игроку

- `/axiom admin cleanup` - очистка неактивных наций

## Права (Permissions)

### Базовые права
- `axiom.use` - использовать основные функции
- `axiom.nation.use` - использовать национальные функции
- `axiom.economy.use` - использовать экономические функции
- `axiom.diplomacy.use` - использовать дипломатические функции

### Продвинутые права
- `axiom.admin` - все права администратора
- `axiom.nation.admin` - административные функции наций
- `axiom.economy.admin` - административные функции экономики
- `axiom.war.admin` - административные функции войны
- `axiom.testbot.use` - использовать автобота

### Права просмотра
- `axiom.economy.see` - видеть чужие балансы
- `axiom.nation.see` - видеть информацию о чужих нациях
- `axiom.stats.see` - видеть статистику сервера
