## 🚀 Готово к использованию

### Установка
```bash
cd axiom-mod-integration
./gradlew build
# JAR в build/libs/axiomui-0.1.0.jar
```

### Использование
```
1. Скопировать JAR в mods/
2. Запустить Minecraft 1.20.1 с Forge
3. Нажать F1 или /axiomui
4. Наслаждаться!
```
