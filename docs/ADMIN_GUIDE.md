# Руководство администратора

Здесь описано, как обновлять файлы, которые скачивают игроки через лаунчер.

## Сервер обновлений
Адрес: `http://193.23.201.6:8080/updates/`
Путь на VPS: `/var/www/axiom/updates`

Лаунчер скачивает файл `client_core.zip`.

## Как выпустить обновление клиента

Если вы изменили конфиги, добавили библиотеки или обновили версию Forge, нужно обновить `client_core.zip` на сервере.

**Инструкция:**

1.  Подготовьте "эталонный" клиент в папке `build_portable/AxiomClient/minecraft/`.
    *   В ней должны быть папки `versions`, `libraries`, `assets`, `config`.
    *   Убедитесь, что игра запускается из этой папки.
2.  Запустите скрипт обновления:
    ```bash
    bash upload_client_core.sh
    ```
3.  Скрипт автоматически:
    *   Запакует нужные папки в архив.
    *   **Исключит** папку `mods` (моды игроки ставят сами или через отдельный архив, если вы решите изменить логику).
    *   Загрузит архив на VPS.

## Управление сервером

Для управления файлами на VPS (заливка модов, ядра сервера и т.д.) используйте интерактивное меню:
```bash
./update-vps.sh
```

## Запуск игрового сервера

Сервер находится в папке `server/`.
Запуск:
```bash
./server/start.sh
```
