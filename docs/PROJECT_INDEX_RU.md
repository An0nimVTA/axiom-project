# Индекс документации AXIOM

Короткий навигатор по ключевым документам проекта и месту, где лежит важная информация.

## Что это за проект
AXIOM — военно‑политико‑индустриальный серверный проект (Minecraft 1.20.1), где ядро отвечает за нации, экономику, дипломатию, войны, технологии и территориальный контроль.

## Стек (текущее состояние)
- Сервер: Mohist 1.20.1 (Forge 47.4.4 + Bukkit/Spigot API).
- Плагин: Java (target 17), сборка Maven.
- UI‑мод: Forge 1.20.1, сборка Gradle.
- Лаунчер: Kotlin/JVM, сборка Gradle.
- Конфиги/данные: YAML и JSON.
- Автотесты: Python + shell-скрипты.
