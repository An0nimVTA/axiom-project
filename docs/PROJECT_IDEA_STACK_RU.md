# Идея проекта и стек

## Идея проекта
AXIOM — военно‑политико‑индустриальный сервер для Minecraft 1.20.1.  
Цель: дать игрокам понятный путь прогрессии от раннего этапа (база, ресурсы, первичные технологии) к зрелому государству с экономикой, дипломатией, войнами, инфраструктурой и контролем территорий.

Ключевые механики:
- Нации и управление территориями (по чанкам).
- Экономика и индустрия (ресурсы, переработка, цепочки поставок).
- Политика и дипломатия (союзы, санкции, войны, перемирия).
- Технологии и прогрессия стадий (ранняя/средняя/поздняя).
- Военная система (стадии войны, завоевания, условия победы).
- UI‑мод для удобного доступа к меню и картам.

## Технический стек (текущее состояние)
- Сервер: Mohist 1.20.1 (Forge 47.4.4 + Bukkit/Spigot API).
- Плагин: Java (target 17), сборка Maven.
- UI‑мод: Forge 1.20.1, сборка Gradle.
- Лаунчер: Kotlin/JVM, сборка Gradle.
- Конфиги/данные: YAML и JSON.
- Автотесты: Python + shell‑скрипты.
