# Инструкция пользователя (Axiom Client)

## Установка и запуск

Клиент Axiom является **Portable** (переносным). Это значит, что он не требует установки в систему и хранит все свои файлы в одной папке.

### Шаг 1: Подготовка
Убедитесь, что у вас есть папка клиента (например, `AxiomClient`). В ней должны быть:
*   `AxiomLauncher.exe` (Лаунчер)
*   Папка `runtime/` (В ней должна лежать Java)
*   Папка `minecraft/` (Файлы игры)

### Шаг 2: Запуск (Windows)
Просто запустите файл `AxiomLauncher.exe`.

*   Лаунчер автоматически проверит наличие обновлений на сервере.
*   Если вышли обновления ядра или конфигов — они скачаются автоматически.
*   **Важно:** Лаунчер НЕ трогает ваши моды в папке `minecraft/mods`, он обновляет только ядро игры.
*   Игра запустится и автоматически подключится к серверу.

### Шаг 2: Запуск (Linux)
Откройте терминал в папке клиента и запустите:
```bash
./start.sh
```

## Решение проблем

**Черное окно и закрытие:**
Проверьте, что в папке `runtime` действительно есть Java. Путь должен быть `runtime/bin/java`.

**Игра не запускается:**
Возможно, повреждены файлы в папке `minecraft`. Попробуйте удалить всё из папки `minecraft`, кроме папки `mods`, и запустить лаунчер снова.
