ИНСТРУКЦИЯ ДЛЯ AI (<=1000 символов)
Цель: создать графику карточек для UI. Используй списки ID из UI_CARD_IDS.txt и требования из UI_CARD_ASSET_SPEC.txt. Формат: PNG 32‑bit RGBA, прозрачный фон. Имена файлов: card_<menu>_<id>_{normal|hover|pressed|disabled}.png. Размеры: основная 420x240, узкая 420x140, маленькая 280x140. Поля 20px. Иконка 64x64 слева сверху. Заголовок ≤18 символов, описание 1–2 строки (26–28 символов на строку, разделитель “|”). Состояния: normal (база), hover (ярче + тень), pressed (темнее + смещение 2–4px), disabled (40–60% прозрачность + серый фильтр). Для динамических списков сделай универсальные шаблоны: card_dynamic_list_{normal|ok|warning|critical}.png. Не рисуй текст в шаблонах, если он будет подставляться сервером/модом.
