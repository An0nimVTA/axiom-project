# AXIOM Геополитический Движок - Полная Установка

## Описание
Полный комплект установки AXIOM Geopolitical Engine с интеграцией модов, веб-интерфейсом и всеми системами.

## Содержимое Пакета

### 1. Основной плагин AXIOM
- **axiom-plugin/** - Ядро системы с 165+ системами
- **Mod Integration System** - Интеграция 30+ модов
- **Recipe Integration** - Кросс-модовые рецепты
- **Technology Tree** - 5-ступенчатое развитие (Каменный век → Военно-промышленный комплекс)
- **Nation System** - Полная система наций с экономикой и дипломатией
- **Military System** - Войны, осады, захваты, дипломатия

### 2. WEB Интерфейс
- **Website/** - Реакт-приложение для админки
- **Интерактивные карты** - Визуализация территорий
- **Статистика в реальном времени** - Экономика, военная сила, технологии
- **Управление модами** - Балансировка и мониторинг

### 3. Системы баланса
- **ModPackManagerService** - Управление пакетами модов
- **DynamicModBalancerService** - Динамическое балансирование
- **RecipeIntegrationService** - Интеграция рецептов
- **ModIntegrationEnhancementService** - Расширенная совместимость

## Архитектура 5-Ступенчатой Прогрессии

1. **Каменный Век** (Tier 1) - Выживание, базовые инструменты
2. **Промышленная Революция** (Tier 2) - Энергия, автоматизация, простые машины
3. **Эпоха Накопления** (Tier 3) - Хранение, логистика, массовое производство
4. **Высокие Технологии** (Tier 4) - Оптимизация, продвинутые системы
5. **Военно-Промышленный Комплекс** (Tier 5) - Пик развития, вооружения, ракеты

## Требования

### Сервер
- **Java**: 17+
- **Minecraft**: 1.20.1
- **Mohist**: 1.20.1 (рекомендуется для модов) или Paper 1.20.1
- **RAM**: 8ГБ+ (16ГБ+ рекомендуется)
- **Моды**: 30+ модов (см. INSTALLATION.md)

### Поддерживаемые Моды
- **Тактические**: TACZ, PointBlank, Ballistix, Superb Warfare
- **Промышленные**: Industrial Upgrade, Immersive Engineering, Applied Energistics 2
- **Энергетические**: Mekanism, Thermal, Quantum Generators
- **Логистические**: Simply Quarries, Immersive Vehicles, AshVehicle
- **Оптимизационные**: Embeddium, Entity Culling, FerriteCore

## Установка

### 1. Подготовка сервера
```
# 1. Установите Mohist 1.20.1 сервер
# 2. Поместите в папку mods требуемые моды
# 3. Поместите axiom-plugin.jar в папку plugins/
# 4. Запустите сервер один раз для генерации конфигов
# 5. Остановите сервер
# 6. Отредактируйте config.yml
# 7. Запустите сервер снова
```

### 2. Конфигурация
Редактируйте следующие файлы:
- `plugins/AXIOM/config.yml` - Основная конфигурация
- `plugins/AXIOM/modpacks.yml` - Пакеты модов
- `plugins/AXIOM/recipe-integration.yml` - Интеграция рецептов
- `plugins/AXIOM/dynamic-balancing.yml` - Динамический баланс

### 3. Запуск веб-интерфейса
```
cd website
npm install
npm run build
npm start
```

## Команды

### Основные команды
- `/axiom` - Главное меню AXIOM
- `/nation create <name>` - Создать нацию
- `/claim` - Захватить чанк
- `/unclaim` - Освободить чанк
- `/technology` - Дерево технологий

### Команды модов
- `/modpack list` - Список пакетов модов
- `/recipeint stats` - Статистика интеграции рецептов
- `/modbal global` - Глобальный баланс модов
- `/modbalancer status` - Статус динамического баланса
- `/dmb stats` - Статус динамической балансировки

### Административные команды
- `/axiom modenhance [параметры]` - Управление интеграцией модов
- `/axiom recipeint [параметры]` - Управление рецептами
- `/axiom modbalancer [параметры]` - Управление балансом

## Конфигурация

### Основные параметры
```yaml
modIntegration:
  enabled: true
  modDetectionIntervalMinutes: 30
  energySystem:
    standardUnit: "FE"
    conversionRates:
      rf: 1.0  # Redstone Flux
      eu: 0.25  # IC2 Energy (4 FE = 1 EU)
```

### Интеграция рецептов
```yaml
recipeIntegration:
  enabled: true
  balancing:
    maximum_recipe_cost_multiplier: 2.0
    minimum_resource_efficiency: 0.5
    cross_mod_synergy_bonus: 0.15
    conflicting_mod_penalty: 0.2
```

## API и интеграция

### REST API
- `GET /api/nations` - Список всех наций
- `GET /api/territory` - Информация о территориях
- `GET /api/mods` - Статистика модов
- `GET /api/economy` - Экономическая информация
- `POST /api/command` - Выполнение команд

### WebSocket Events
- `status_update` - Обновление статуса сервера
- `nation_created` - Создание новой нации
- `war_declared` - Объявление войны
- `technology_unlocked` - Открытие технологии

- **API ключи** для защиты веб-интерфейса

## Устранение неполадок

1. **Плагин не загружается**: Проверьте зависимости (WorldEdit)
2. **Моды не обнаружены**: Убедитесь в совместимости версий
3. **Проблемы с производительностью**: Увеличьте RAM или отрегулируйте частоты обновления
4. **Проблемы с интеграцией**: Проверьте настройки в config.yml
