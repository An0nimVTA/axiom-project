# AXIOM Geopolitical Engine

Комплексный геополитический и экономический движок для Minecraft серверов с более чем 165 системами и возможностями.

## Описание

AXIOM - это мощный плагин, который превращает обычный Minecraft сервер в глубокий симулятор геополитики, экономики и дипломатии. Плагин включает в себя более 165 различных систем и сервисов, охватывающих все аспекты государственного управления и международных отношений.

## Особенности

### Национальная система
- Создание наций с уникальными именами и флагами
- Захват и управление территорией
- Внутреннее администрирование (города, законы, налоги)

### Экономика
- Финансовая система (валюты, банки, кошельки)
- Торговля и рынки
- Фондовая биржа
- Система налогообложения

### Дипломатия
- Договоры и соглашения
- Союзы и вассалитет
- Войны и мир
- Дипломатические миссии

### Военные системы
- Объявление и ведение войн
- Мобилизация и военные ресурсы
- Осады и штурмы
- Шпионаж и разведка

### Социальные системы
- Религии и культы
- Образование и наука
- Культура и нации
- Система выборов

### Инфраструктура
- Транспортные системы
- Здания и сооружения
- Ресурсы и производство
- Торговые пути

### И многое другое...
- 165+ различных систем и сервисов
- Интеграция с модами
- Автосохранение данных
- Multi-world поддержка
- Поддержка PlaceholderAPI

## Команды

### Основные команды
- `/axiom help` - основное меню и список команд
- `/nation create <название>` - создать нацию
- `/nation claim` - захватить территорию
- `/nation unclaim` - освободить территорию
- `/nation list` - список всех наций

### Экономические команды
- `/wallet` - проверить баланс
- `/wallet pay <игрок> <сумма>` - перевод денег
- `/bank` - банковская система
- `/stock` - фондовая биржа

### Дипломатические команды
- `/treaty` - дипломатические соглашения
- `/ally` - создание союзов
- `/war` - объявление войны
- `/diplomacy` - дипломатическое меню

## Требования

- Minecraft 1.20.1
- Spigot/Paper 1.20.1+ (рекомендуется)
- Java 17+
- (Опционально) Mohist (для интеграции с модами)

## Установка

1. Скачайте последнюю версию плагина из `target/` после компиляции
2. Поместите JAR файл в папку `plugins/` вашего сервера
3. Перезапустите сервер
4. Первый запуск создаст все необходимые файлы конфигурации
5. При необходимости отредактируйте `plugins/AXIOM/config.yml`

## Конфигурация

После первого запуска создается файл `plugins/AXIOM/config.yml` с настройками:
- Настройки экономики
- Настройки наций
- Настройки защиты
- Настройки производительности
- Настройки автосохранения

## Автотесты

Плагин включает систему автоматического тестирования. Подробности в файле [AUTOBOT.md](AUTOBOT.md).

## API

Плагин предоставляет API для других плагинов:
- PlaceholderAPI интеграция
- Vault интеграция
- Event API
- Data API
