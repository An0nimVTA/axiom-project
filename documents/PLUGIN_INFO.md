# Плагин AXIOM Geopolitical Engine

## Описание
Плагин AXIOM - это комплексная система геополитического симулятора для Minecraft серверов, включающая более 165 различных систем и сервисов.

## Структура плагина
```
axiom-plugin/
├── src/                    # Исходный код Java
│   └── main/java/com/axiom/
│       ├── command/        # Команды плагина
│       ├── expansion/      # Расширения (PlaceholderAPI)
│       ├── gui/            # GUI меню
│       ├── listener/       # Слушатели событий
│       ├── model/          # Модели данных
│       └── service/        # Бизнес-логика (165+ сервисов)
├── pom.xml                # Maven конфигурация
├── plugin.yml             # Конфигурация плагина
├── README.md              # Документация
├── AUTOBOT.md             # Инструкция по автотестам
└── target/                # Скомпилированный JAR файл
```

## Системы и сервисы (основные)
- Управление нациями и территориями
- Экономическая система
- Дипломатия и международные отношения
- Религиозные системы
- Военные и боевые системы
- Торговля и рынки
- Образование и культура
- Экология и ресурсы
- Система событий и новостей
- Модпаки и интеграция с модами

## Требования
- Java 17+
- Maven 3.6+
- Spigot/Paper 1.20.1+
- (Опционально) Mohist (Forge + Bukkit)

## Компиляция
```bash
mvn clean package
```

Скомпилированный файл будет в `target/axiom-geopolitical-engine-1.0.0.jar`

## Установка
1. Поместить JAR файл в папку `plugins/` сервера
2. Перезапустить сервер
3. Первый запуск создаст конфигурацию в `plugins/AXIOM/`
4. Отредактировать `config.yml` при необходимости
